## Features
- Home page with product listings
- Product detail page
- Shopping cart with local storage
- User registration and login
- Role-based access (user/admin)
- Admin panel for managing products
- Responsive design
- Banner slider + flash sale section
- Favorite products (localStorage)
- Dynamic category pages (`category.html?cat=...`)
- Search history + autocomplete
- Price filter / sort by price
- JWT auth support stubs (khi có backend API)

## How to Run
Open `index.html` in your web browser to view the website.

For a local server, you can use Python if installed: `python -m http.server` and visit http://localhost:8000.

## Notes
- Images are online placeholders; replace with actual product images.
- User data and products are stored in localStorage.
- No backend; this is a frontend-only demo.
- Register as admin to access admin panel.
